Code Book

This Code Book summarizes the resulting data fields from the activity and training tracking modules.

The source data can be found here: http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

Rows - Description of code:
1:8 - Install the necessary packages and set the working directory.
10:18 - read all the CSVs into the IDE.
20:27 - Add column names to the base raw data that you need for joining tables later.
29:32- Create a subset with only the relevant columns (mean and St. Dev)
34:49 - Loop through the junk characters in the column headers, and clean them up
50:51 - Join with the table to organize all the variables with the factor Activity Type
53:54 - Organize the final tables of tidy raw data and averages of all the variables' averages for each Activity Type
56:58 - Export the two tables to text files.
